# Veri Görselleştirme Ödevi

Bu projede verilen veri seti kullanılarak temel veri görselleştirme çalışmaları yapılmıştır.

## İçerik
- Veri setinin incelenmesi
- Scatter plot, bar chart ve temel grafikler
- Matplotlib ve Seaborn kullanımı

Bu çalışma Veri Bilimine Giriş bitirme ödevi kapsamında hazırlanmıştır.
